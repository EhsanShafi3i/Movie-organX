# MOVIE OrganX

An app to organize movies by year of production
## Installation

First you have to install IMDbPY

```bash
pip install IMDbPY
```
And clone the repo 
```bash
git clone https://github.com/EhsanShafi3i/Movie-organX.git
```
## Usage
After cloning\
### Windows
```bash
 cd .\Movie-organX\
```
```bash
python main.py
```
### Linux
```bash
cd Movie-organX/
```
```bash
python3 main.py
```
